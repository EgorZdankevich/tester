    Terminal Linux
1. Посмотреть где я - pwd
2. Создать папку - mkdir group_20
3. Зайти в папку - cd group_20
4. Создать 3 папки - mkdir g_1 g_2 g_3
5. Зайти в любую папку - cd g_1
6. Создать 5 файлов (3 txt, 2 json) - touch gg_1.txt gg_2.txt gg_3.txt gg_4.json gg_5.json
7. Создать 3 папки - mkdir ggg_1 ggg_2 ggg_3
8. Вывести список содержимого папки - ls -la
9. Открыть любой txt файл - vim gg_1.txt
10. Написать туда что-нибудь, любой текст - text_text_text
11. Сохранить и выйти - :wq
12. Выйти из папки на уровень выше - cd ..
13. Переместить любые два файла - mv g_1/gg_1.txt g_2/gg_1.txt
                                  mv g_1/gg_2.txt g_2/gg_2.txt
14. Скопировать любые два файла - cp g_1/gg_3.txt g_2/gg_3.txt
                                  cp g_1/gg_4.json g_2/gg_4.json
15. Найти файл по имени - find gg_3.txt
16. Посмотреть содержимое в реальном времени (команда grep) - tail -f gg_1.txt
17. Вывести несколько первых строк из текстового файла - head -n3 gg_1.txt
18. Вывести несколько последних строк из текстового файла - tail -n4 gg_1.txt
19. Просмотреть содержимое длинного файла (команда less) - less gg_1.txt
20. Вывести дату и время - date
